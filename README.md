# machine-learning
ML kodları ve Python projeleri

Bu repository modüler bir yapıda kurgulanmıştır. Her bir ana başlık (ANN, Q-Learning, SVM vb.) kendi klasörü içerisinde yer almaktadır. 

Önemli Not: Projelerin teknik detayları, matematiksel arka planları ve nasıl çalıştırılacaklarına dair kapsamlı bilgiler, her klasörün içerisinde yer alan özel README dosyalarında ayrı ayrı belirtilmiştir.

--- Neler Var?

Sınıflandırma Modelleri, Ensemble & Boosting, Regresyon Analizleri, Gözetimsiz Öğrenme, Takviyeli Öğrenme, Model Optimizasyonu

--- Teknolojik Stack

Diller ve Frameworkler: Python, TensorFlow, Keras.

Veri Bilimi Araçları: Scikit-learn, Pandas, NumPy, SciPy.

Görselleştirme: Matplotlib, Seaborn.

--- Nasıl İncelenmeli?

İlgilendiğiniz algoritmanın klasörüne giderek o klasöre özel hazırladığım README dosyasını okuyabilir, kod blokları ve performans çıktıları üzerinden projenin detaylarına hakim olabilirsiniz.
